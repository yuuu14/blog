## Boyer-Moore Majority Voting Algorithm
get majority element: element in array with size of N appearing more than $\bigg\lfloor \frac{N}{2} \bigg\rfloor$

```python
cnt, cand = 0, 0
for n in nums:
    if cnt == 0:
	    cand = n
    if cand == n:
	    cnt += 1
    else:
	    cnt -= 1

```
